## Списки 

Чтобы добавить не нумерованные списки надо добавить звездочку (*).
нпример 
* Элемент 1
* Элемент 2
* Элемент 3
Чтлбы добавить нумерованные пункты просто их пронумеровать 
например 
1. первый пуект
2. второй пункт
